# curso-apache-kafka-spring
Projeto para aprendizado do Apache Kafka

Neste projeto foi desenvolvido o microservico de pagamento, onde se prouduzia uma mensagem para o Kafka, com algumas informações da compra de um produto
ficticio. O outro microservico é o consumidor, que fica aguardando o objeto que foi produzido e enviado ao Kafka, pra que se possa tomar ações após a mensagem recebida.

Tecnologias usadas:
- Apache Kafka
- Spring Boot
- Docker
- Docker Compose
- Git para versionamento do código implementado
- Github para subir o projeto
